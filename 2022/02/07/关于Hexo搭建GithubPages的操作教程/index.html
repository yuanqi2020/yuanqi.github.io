    关于Hexo构建Github Pages操作

    一、创建github

    1、创建库时，最好以用户名作为仓库名称，如图所示

    2、全局关联Github账号

    连接测试ssh -T git@github.com

    如果返回用户名称，则说明ok了。

    二、hexo安装

    安装npm、node.js、hexo

    安装所需要的框架

    Mac环境下，安装homebrew

    brew install npm

    如果版本不对，请升级npm版本

    三、发布hexo

    新建一篇文章

    1)进入blog文件夹

    请注意，需要写上双引号，不然会报错。

    提交更新

    如果报错[hexo d命令报错 ERROR Deployer not found: git]，未安装git，则提交

    五、日常更新

    1、进入文件夹~/yuanqi/blog

    2、hexo new "文章title"

    3、hexo g

    4、hexo d
